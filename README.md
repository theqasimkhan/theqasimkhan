## &nbsp; Featured Projects

- ### 🧠 &nbsp; [CliniSynapse AI](https://github.com/theqasimkhan/CliniSynapse-AI) &nbsp;—&nbsp; *Patient-Centric RAG Clinical Decision Support System*

  A production-grade AI platform for differential diagnosis. Retrieves from **167,000 real-world PMC patient reports** using hybrid dense + sparse retrieval, re-ranks with a cross-encoder, and synthesises structured clinical reasoning via Gemini and OpenAI — with explicit hallucination checks and confidence scoring.

  **Real-world Impact** → Bridges the gap between LLM hallucination and evidence-based clinical reasoning. Every output is grounded in real patient data, not model inference alone.

- ### 🔭 &nbsp; [RepoMind AI](https://github.com/theqasimkhan/RepoMind-AI) &nbsp;—&nbsp; *GitHub Repository Intelligence Platform*

  **Real-world Impact** → Turns any codebase into a queryable knowledge base — onboarding, audit, and architecture review in seconds, not hours.

- ### 🫁 &nbsp; [MultiModal AI — Chest X-ray](https://github.com/theqasimkhan/MultiModal-AI-for-Chest-Xrays) &nbsp;—&nbsp; *Automated Radiology with CV + NLP*

  Multimodal medical AI combining Computer Vision and NLP for automated radiology. Detects five critical pulmonary pathologies via **DenseNet-121**, then synthesises professional diagnostic reports and answers clinical Q&A with **Flan-T5**. Complete Flask web interface with patient history management.

  **Real-world Impact** → Demonstrates how CV + NLP fusion can automate the first-pass diagnostic workflow for radiologists in resource-constrained settings.

- ### 🔍 &nbsp; [Enhanced CBIR — Visual Search Engine](https://github.com/theqasimkhan/Enhanced_CBIR_System) &nbsp;—&nbsp; *High-Performance Content-Based Image Retrieval*

  Extracts **138,923 multi-modal features** per image — HOG, LBP, ORB keypoints, colour histograms, and Hu moments — compressed to optimal dimensions with PCA and matched across 6 similarity metrics. Parallel processing delivers a measured **1.74× throughput improvement** with production-ready error handling and a full test suite.

  **Real-world Impact** → Demonstrates from first principles how modern visual search engines work — relevant to e-commerce, medical imaging, and digital asset retrieval at scale.
